# Movie Ratings Analysis
A comprehensive data analysis project exploring movie ratings, genres, and trends using Python, Pandas, and Matplotlib.

# Project Overview
1. Dataset: 20 movies across multiple genres (1993-2019)
2. Analysis: 8 key questions about movie ratings and popularity
3. Tools: Python, Pandas, Matplotlib
4. Skills: Data cleaning, visualization, statistical analysis

# Analysis Questions
1. Top 5 highest-rated movies
2. Average rating per genre
3. Movies released per year
4. Highly-rated popular movies (rating > 8.0 & votes > 50,000)
5. Genre with highest average votes
6. Longest movie by duration
7. Bar chart: Genre vs Average Rating
8. Line plot: Movies released per year

# Technologies Used
1. Python 3
2. Pandas - Data manipulation and analysis
3. Matplotlib - Data visualization
4. NumPy - Numerical operations
